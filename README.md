### 상단 Follow
Follow 페이지로 이동 후 검색창에 유저 아이디를 입력하면 실시간으로 검색 

### 하단 더보기
더보기 버튼을 클릭시 다음 게시물 로드

### 하단 + 
+ 버튼 클릭시 파일업로드 팝업이 나오고 이미지 선택시 필터페이지로 이동
필터를 선택 후 Next 버튼을 클릭하면 글작성 페이지로 이동 Posting을 누르면
처음페이지에 맨 위 게시물로 생성
